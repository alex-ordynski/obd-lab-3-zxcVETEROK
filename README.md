# Ласкаво просимо до Гоґвортсу!

Цей репозиторій містить усе необхідне для виконання завдань.

### Що в цьому репозиторії?

* **Середовище Codespaces:** Повністю налаштоване середовище для розробки, яке працює у вашому браузері.
* **Інструменти:** Вже встановлено **Rust**, **SQLite** та **NASM**.
* **База даних:** Файл `hogwarts.db` створюється автоматично. Він містить таблиці `wizards` (чарівники) та `houses` (гуртожитки) з початковими даними.

### Як почати?

1.  Натисніть зелену кнопку **`< > Code`** вгорі сторінки.
2.  Перейдіть на вкладку **"Codespaces"**.
3.  Натисніть **"Create codespace on main"**, щоб запустити ваше робоче середовище.
